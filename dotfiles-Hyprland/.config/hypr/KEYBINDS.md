# Hyprland Keybinds

Super = tecla Windows (ou Command no Mac)

**Super + K** - Abrir este tutorial

## Aplicativos

| Teclas              | Acao                    |
|---------------------|-------------------------|
| Super + Enter       | Terminal (Kitty)        |
| Super + Shift + Enter | Terminal flutuante    |
| Super + D           | Launcher                |
| Super + E           | Thunar (arquivos)       |
| Super + A           | Dashboard               |
| Super + M           | Music panel             |
| Super + X           | Travar tela (hyprlock)  |
| Menu                | Define (dicionario)     |

## Janelas

| Teclas                | Acao                |
|-----------------------|---------------------|
| Super + Shift + Q     | Fechar janela       |
| Super + Shift + Espaco| Alternar flutuante  |
| Super + F             | Tela cheia          |
| Super + W             | Alternar wallpaper  |
| Super + Shift + W     | Wallpaper aleatorio |
| Super + P             | Pseudo               |
| Super + J             | Alternar split       |

## Navegacao

| Teclas                 | Acao                  |
|------------------------|-----------------------|
| Super + setas          | Mover foco entre janelas |
| Super + Shift + setas  | Mover janela          |
| Super + Ctrl + setas   | Redimensionar janela  |
| Super + Tab            | Proxima janela        |
| Super + Shift + Tab    | Janela anterior       |
| Super + 1 a 5          | Ir para workspace 1-5 |
| Super + Shift + 1 a 5  | Mover janela para workspace |
| Super + scroll mouse   | Trocar workspace      |
| Super + S              | Workspace especial    |
| Super + Shift + S      | Mover para especial   |

## Notificacoes

| Teclas              | Acao                  |
|---------------------|-----------------------|
| Super + N           | Abrir notificacoes    |
| Super + Shift + N   | Fechar notificacoes   |

## Media

| Teclas              | Acao                  |
|---------------------|-----------------------|
| F1                  | Mutar audio           |
| F2                  | Diminuir volume       |
| F3                  | Aumentar volume       |
| Print               | Screenshot            |
| Shift + Print       | Screenshot de area    |
| Super + R           | Iniciar gravacao      |
| Super + Shift + R    | Parar gravacao        |

## Mouse

| Acao                         | Funcao           |
|------------------------------|------------------|
